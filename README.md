# Dark Mode Extension

A simple, customizable dark mode browser extension designed for personal use. Instantly toggle a dark theme for any website to reduce eye strain and make your browsing experience more comfortable.

## Features

- Enable and disable dark mode with one click
- Lightweight and privacy-friendly (no data collection)
- Adjustable to fit your visual preferences
- Works across most popular web browsers

## Installation

1. Clone or download this repository to your computer.
2. Open your browser’s extensions page:
    - **Chrome**: `chrome://extensions/`
    - **Firefox**: `about:addons`
    - **Edge**: `edge://extensions/`
3. Enable "Developer mode" or "Load unpacked" (depending on browser).
4. Select the extension folder to load it.

## Usage

- Click the extension icon in your browser to toggle dark mode on or off.
- The extension will automatically apply a dark theme to the current website.

## Customization

You can modify the CSS in the extension’s files to suit your needs. Check the `content-script.css` or similar stylesheet file.
